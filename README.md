#  Uber Rides Data Analysis Dashboard
   Live Dashboard Link: https://uber-rides-analysis-by-abhi.streamlit.app/

## Project Overview

This project analyzes a synthetic Uber ride-booking dataset containing **150,000 ride records** from the Delhi-NCR region for the year 2024. The objective is to extract meaningful business insights related to ride demand, revenue, customer experience, cancellations, and operational performance.

The project demonstrates the complete analytics workflow—from data cleaning and feature engineering using **Python and Pandas** to visualization with **Matplotlib** and an interactive dashboard built using **Streamlit**.

---

## Objectives

* Analyze ride booking patterns across different months, locations, vehicle types, and time slots.
* Measure business KPIs such as revenue, booking volume, completion rate, and cancellation rate.
* Understand customer and driver cancellation behavior.
* Identify operational bottlenecks affecting ride completion.
* Build an interactive dashboard for business decision-making.

---

## Dataset

The dataset contains **150,000 Uber ride booking records** with information such as:

* Booking Date & Time
* Pickup & Drop Locations
* Vehicle Type
* Booking Status
* Booking Value (Fare)
* Payment Method
* Customer Rating
* Driver Rating
* Average Waiting Time
* Driver Cancellation Reasons
* Customer Cancellation Reasons
* Incomplete Ride Reasons

## Key Business Insights

* Morning and Evening are the peak demand periods with the highest number of completed rides.
* Driver cancellations contribute the largest share of unsuccessful bookings.
* UPI is the most preferred payment method.
* Auto is the most frequently booked vehicle type and generates the highest revenue.
* Several pickup locations consistently contribute the highest revenue, indicating high-demand zones.

---

## Business Recommendations

* Increase driver availability during Morning and Evening peak hours.
* Reduce driver cancellations using incentive and performance programs.
* Improve driver allocation to reduce "No Driver Found" bookings.
* Focus promotional campaigns on high-demand locations.
* Continue supporting digital payments, especially UPI.
